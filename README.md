Signea

==========

Sistema acadêmico para gestão de eventos, presenças e certificados do IFFar.

Acesse a aplicação:
https://signea-brown.vercel.app/

Instalação

==========

Dependências
npm install

Desenvolvimento

==========

Inicie o servidor de desenvolvimento:

npm run dev

Build de Produção

==========

npm run build
npm run start

Estrutura do Projeto

==========

src/ — Front-end em React e TypeScript

components/ — Componentes reutilizáveis

pages/ — Páginas principais

public/ — Assets públicos

supabase/ — Configurações e migrations do banco de dados

Backend

==========

O backend utiliza Supabase (Auth, Database e Storage).
As migrations estão na pasta supabase/migrations.

Código não é disponível para comercialização/uso interno sem aviso prévio ou solicitação formal para a equipe.
==========

Projeto acadêmico de uso interno do IFFar – Campus Panambi.
