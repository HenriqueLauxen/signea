Signea

==========

Acesse a aplicação:
https://signea-brown.vercel.app/

Instalação
==========
Dependências
npm install

Desenvolvimento
==========
Inicie o servidor de desenvolvimento:
npm run dev
Build de Produção
==========
npm run build
npm run start


Estrutura do Projeto
==========
src/ — Front-end em React e TypeScript
components/ — Componentes reutilizáveis
pages/ — Páginas principais
public/ — Assets públicos
supabase/ — Configurações e migrations do banco de dados


Backend
==========
O backend utiliza Supabase (Auth, Database e Storage).
As migrations estão na pasta supabase/migrations.
